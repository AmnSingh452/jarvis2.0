# Jarvis AI Chatbot - Demo Video Script

## Video Requirements
- **Duration:** 2-3 minutes maximum
- **Format:** MP4, MOV, or hosted on YouTube/Vimeo
- **Resolution:** 1080p minimum
- **Audio:** Clear narration (optional but recommended)

## Video Script Outline

### Intro (15 seconds)
"Welcome to Jarvis AI Chatbot - an intelligent customer service assistant for Shopify stores."

### 1. App Installation Demo (30 seconds)
- Show app in Shopify admin
- Navigate to app dashboard
- Highlight clean, professional interface

### 2. Widget Configuration (45 seconds)
- Go to Widget Settings
- Show customization options:
  - Color picker (primary/secondary colors)
  - Position settings (bottom-right, bottom-left, etc.)
  - Size options
  - Custom messages
- Save settings and show preview

### 3. Billing Integration (30 seconds)
- Navigate to Billing & Plans
- Show two plans: Essential Chat ($14.99) and Sales Pro ($39.99)
- Demonstrate upgrade process
- Show Shopify billing integration

### 4. Frontend Demo (45 seconds)
- Switch to store frontend
- Show chatbot widget appearance
- Demonstrate chat interaction:
  - Customer asks: "What products do you recommend?"
  - AI responds with product suggestions
  - Show abandoned cart recovery scenario

### 5. Admin Features (30 seconds)
- Back to admin
- Show welcome guide
- Highlight key features:
  - AI-powered responses
  - Product recommendations
  - Abandoned cart recovery
  - Analytics tracking

### Outro (15 seconds)
"Jarvis AI Chatbot - Boost sales with intelligent customer assistance. Install today!"

## Recording Tools
- **Screen Recording:** OBS Studio (free), Camtasia, or Loom
- **Video Editing:** DaVinci Resolve (free), Adobe Premiere, or Canva
- **Hosting:** YouTube (unlisted), Vimeo, or direct upload to app store
